2.1.- Texto en HTML

Encabezados (head):
Menor importancia {h1,h2,h3,h4,h5,h6} Mayor importancia
Parrafos:
Para poder agregar parrafos necesitamos: < p > y < /p >
Salto de linea:
Para realizar un salto de line: < br >
Espacios en blanco:
Para que podamos poner estacios pequeños en blanco es con: &nbsp;
Negritas:
Para que un texto tenga negritas: < b >
Cursivas:
Para que un texto tenga curiva: < i >
Resaltado:
Para que un texto tenga resaltado: < strong >
Enfasis:
Esta etiqueta se utiliza para que semanticamente tenga un alto impacto el texto: < en >
Grupo o selección de texto:
Agrupar un grupo de texto: < span >
Caracteres especiales:
Existen caracteres especiales que para poder insertarlos se necesita, ya sea el nombre o el numero para que se pueda ver en pantalla:
&nombre;
&#numero;
