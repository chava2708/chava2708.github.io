2.2.- Estructura semántica de un documento HTML

Encabezado
< header > : Contiene el título de la página.
Navegación
< nav > : Tiene enlaces para navegar en el sitio web.
Contenido principal
< main > : Todo el cuerpo de la pagina.
Sección
< section > : Se divide en secciones la pagina
Artículo
< article > : Para poder dividir esas secciones en diferentes articulos
Barra lateral
< aside > : El contenido que aparece en la bara lateral del navegador
Pie de página
< footer > : El contenido del pie de pagina
Divisiones
< div > : Para agrupar el contenido
