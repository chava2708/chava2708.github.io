4.4.- DOM (Document Object Model)

DOM

Representa el documento HTML en forma de una estructura de árbol.

El documento se puede manipular fácilmente a través del árbol, es decir, agregar, eliminar, modificar elementos, etc.

  • 1. Acceder elementos del documento.
    • Document
      • .getElementsByTagName(‘etiqueta’)
      • .getElementsByClassName(‘clase’)
      • .getElementById(‘identificador’)
  • 2. Modificar el contenido de un elemento.
    • Element
      • .textContent = ‘texto’
      • .innerHTML = ‘codigo lateral’
  • 3. Agregar elementos (nodos) nuevos.
    1. Crear el elemento (createElement())
    2. Crear el contenido (createTextNode())
    3. Añadir el contenido al elemento (appendChild())
    4. Agregar atributos (setAttribute())
    5. Agregar el nuevo elemento al documento (appendChild())
