Arreglos en JavaScript

  • Creación
    • var arreglo = new Array();
    • var arreglo = [];
    • var arreglo = Array(‘hola’, 123, true);
    • var arreglo = [‘hola’, 123, true];
    • var x = arreglo [0];
    • Arreglo [0] = ‘mundo’;
    • Arreglo [10] = 3456;

    4 5 6
    • var m = [[1,2,3],[4,5,6]]

  • Metodos
    • Concat()-----Concatenar
    • Join()-------Unir todos los elementos de un arreglo en una cadena
    • Push()-------Agregar un elemento al final
    • Pop()--------Eliminar el ultimo elemento del arreglo
    • Reverse()----Regresa un arreglo con los elementos en orden invertido
    • Sort()-------Ordena de forma acendente las cadenas
    • Shift()------Elimina del inicio
    • Unshift()------Inserta al inicio
    • Splice()------Agrega o elimina en cualquier posición
    • Slice()------Extrae un subarreglo
