3.1.- Selectores en CSS

  1. Selector de tipo:
    Selecciona un tipo de elemento HTML y aplica el estilo a todos los elementos del mismo tipo.
    Ejemplo: p { text-align:justify; }

  2. Selector descendente:
    Aplica el estilo a elementos que estan contenidos dentro de otros elementos.
    Ejemplo: p b { color:red; } , ul li b { color:green; }

  3. Selector hijo:
    Similar al descendente pero más restrictivo. Aplica el estilo solamente a los elementos que sean hijos directos de otro elemento.
    Ejemplo: p > b { color:blue; }

  4. Selector por ID:
    Un ID es un atributo de un elemento HTML y se utiliza para aplicar un estilo de manera particular.
    Ejemplo:
    < p id="introduccion">Este es el parrafo de la introducción < /p >
    < p >Este es otro parrafo < /p >
    # introduccion { color:blue; } ( Un selector por id utiliza el simbolo #)

  5. Selector por clase:
    Aplicar el estilo a un conjunto de elementos.
    Ejemplo: < p class="especial" > este es un parrafo especial < /p >

  6. Selector Universal " * ":
    Aplica el mismo estilo a todos los elementos de la pagina.
