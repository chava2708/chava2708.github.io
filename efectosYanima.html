5.2.-Efectos y Animaciones con jQuery

Efectos y Animaciones

  1. Ocultar / Mostrar ( .hide() / .show() / .toggle() )
  2. Fundido gradual ( .fadeOut() / fadeIn() / fadeToggle())
  3. Desplazmiento (.slideUp() / .slideDown() / .slideToggle() )
  4. Animaciones ( .anímate() / .clearQueue() )

Parámetros

  • Sin parámetros
  • Duración del efecto (milisegundos, ‘slow’, ‘fast’ )
  • Suavizado (‘linear, ‘swing’ )
  • Callback (función que se ejecuta al momento de terminar el efecto)
