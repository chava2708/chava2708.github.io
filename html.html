2.- "HTML" HyperText Markup Language" (Lenguaje de Marcado para Hipertextos)

  • Las páginas HTML son documentos de texto.
  • HTML usa elementos para definir la estructura de la página.
  • Los elementos se definen por etiquetas encerradas en paréntesis angulares < y >.
  • El contenido de un elemento se encierra entre etiquetas.
  • Existen etiquetas de apertura <etiqueta> y </etiqueta>.
  • EL nombre de la etiqueta debe ser el mismo y se distingue a la etiqueta de cierre con una diagonal.
    • <etiqueta> contenido </etiqueta>
  • Las etiquetas pueden tener atributos los cuales se especifican en la etiqueta de inicio.
  • Los atributos contienen informacíon adicional acerca del contenido.
    • <etiqueta atributo = " valor " > contenido </etiqueta>
