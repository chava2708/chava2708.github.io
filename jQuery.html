jQuery

"Write less, do more"

John Resig Escribió la libreria de jQuery

Simplifica

  • La interacción con HTML
  • La manipulacion con el DOM
  • La manipulación con CSS
  • Manejo eventos
  • Animaciones
  • Interacciones Ajax

Garantiza la compatibilidad entre navegadores

Todo se realiza a través de una función llamada jQuery()

Recibe como parámetro una expresión CSS

var objeto = jQuery(selector css)

Tipos de Selectorea

  • *
  • Elemento
  • #id
  • .clase
  • sel1,sel2

Filtros Básicos

  • :first  var primero = jQuery(‘*:first’)
  • :last
  • :even-----------posición impar
  • :odd------------posición par
  • :eq(index)-----posición igual que index
  • :gt(index)------mas alla de la posición
  • :lt(index)------antes de una posición
  • :not(selector)
