1.1.- Procedimiento Solicitud/Respuesta del cliente/servidor

Accion Navegador Internet Servidor Disco Duro Servidor
1 Usuario escribe URL
2 DNS busca URL y regresa IP
3 Solicita la página del servidor usando la IP
4 Recibe la Solicitud de la página
5 Obtiene la página del HDD
6 Regresa la página
7 El cliente recibe y muestra la pagina

1.1.1- Procedimiento Solicitud/Respuesta del cliente/servidor en una pagina dinámica

Accion Navegador Internet Servidor Proc PHP Disco Duro Base de Datos
1 Usuario escribe URL
2 DNS busca URL y regresa IP
3 Solicita la página del servidor usando la IP
4 Recibe la Solicitud de la página
5 Obtiene la página
6 Contiene PHP
7 Proceso PHP
8 Ejecutar SQL
9 Recibe los datos
10 Regresa la página
11 El cliente recibe y muestra la pagina
