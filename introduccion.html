1.- Introducción a la web


 "W W W"  World Wide Web   (Red de Alcace Global)

  • Fué creado a principios de 1990´s
  • Se creó por Tim Berners-Lee mejor conocido como el padre de la web

Tim Berners-Lee creó:

  • HTTP (HyperText Transfer Protocol)
  • HTML (HyperText Markup Language)
  • URL (Uniform Resource Locator)
  • El primer navegador WorldWideWeb
  • El primer servidor "HTTPD"
