2.4.- Tablas en HTML

Elementos de una Tabla

Tabla
< table > : Para indicar que se va a iniciar una tabla
Descripcion
< caption > : Para agregar una breve descripcion de la tabla
Encabezado
< thead > : Para indicar los encabezados
Pie de pagina
< tfoot > : Para indicar el pie de la tabla
Cuerpo de la pagina
< table > : Donde va todo la tabla
Fila
< tr > (table row) : Para indicar las filas
Celda de encabezado
< th > : Para indicar el nombre de la celda
Celda de cuerpo y pied
< td > : Para indicar la celda del cuerpo y pie
