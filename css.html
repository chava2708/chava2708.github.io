3.0.2.- Reglas de CSS

Las reglas permiten cambiar la apariencia(presentacion) de los elementos en HTML

Sintaxis general de css:
Selector { declaraciones }
Declaraciones:
propiedades : valor ;

Ejemplos:

  • h1(selector) {color(propiedad) : blue(valor)};
  • p{font-size : 200% ;}

Comentarios:

  • Para hacer comentarios en codigo de css " /* " y " /* "

3.0.3.- Tipos de estilo

Tipos de estilo:

  1. Por default(Navegador)
  2. Del usuario(Archivo local)
  3. Hoja de estilos externa: Utiliza la etiqueta < link > para hacer referencia al archivo
  4. Estilo interno: Utiliza la etiqueta < Style >
  5. Estilo en linea
Archivo local (estilos.css):
h1 { color : blue ; }

Hoja de estilo externa:
< link href = "estilos.css" type = "text/css" ref = "style sheet" >

Estilo interno:
< style > h1 { color : blue ; } < /style >

Estilo en linea:
< h1 style = "color : blue;" > titulo < /h1 >
