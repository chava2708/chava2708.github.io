2.3.- Enlaces, Imagenes y Listas en HTML

Enlaces

En HTML existe 3 tipos de enlaces y se usan con la etiqueta: < a >

  1. Relativo hace referencia al mismo sitio
  2. Absoluto hace referencia a otra pagina
  3. Contenido para descargar
Ademas lleva atributos como por ejemplo:
  • href= "enlace" para asignar algun elace
  • target= "_blank (abre otra pestaña del navegador) o _serfl (abre en la misma pestaña)

Ejemplo hecho en clase:Enlaces.

Imagenes

Para poder agregar imagenes es con la etiqueta < img >

Las imagenes llevan varios atributos que son los siguientes:

  • scr = "Nombre del archivo"
  • height = "Altura"
  • widht = "Ancho"
  • alt = "Texto alternativo"

Ejemplo hecho en clase:Imagenes.

Listas

Para poder agregar listas al documento html es con las siguientes etiquetas:

  • Listas odenadas < ol > (ordered list)
  • Listas no odenadas < li > (unordered list)
  • Lista de descripcion < dl > (description list)

Los elementos de la lista Ordenada y No Ordenada se especifican por la etiqueta

< dt > (description term)

Los elementos de la lista de descripcion utiliza las etiquetas

< dd > (description data)

Ejemplo hecho en clase:Listas.
