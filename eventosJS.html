Pasos para el manejo de eventos

  1. Seleccionar el elemento (nodo)
  2. Indicar el evento en el elemento seleccionado
  3. Escribir el script a ejecutar en respuesta al evento

Tipos de manejadores

  • 1. Manejador HTML
    • < elemento onevento = "código"> CONTENID </elemento>
    • < a onclick = “alert(‘hola’)”> Saluda </a>
  • 2. Manejador DOM tradicional
    • elemento.onevento = nombreFunción
    • documento.getElementById('enlace').onclick = alert('Hola')

    • Desventaja: un elemento un evento

  • 3. Escuchador DOM nivel 2
    • Elemento.addEventListener(‘evento’,nombreFuncion)
    • document.getElementById(‘enlace’).addEventListener(‘click’,saludar)
